A Tumblr-style askbox for WordPress.

== Description ==

A Tumblr-style askbox for WordPress. Shortcode is `[askbox]`, with asks stored as post drafts. That's it.

== Installation ==

1. Upload the `askbox` folder to the `/wp-content/askbox/` directory.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Add the `[askbox]` shortcode wherever you want your askbox to appear.
